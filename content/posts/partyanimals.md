## 通用型

| 说明   | 快捷键         |
| ------ | -------------- |
| 保存   | `ctrl+s`       |
| 复制   | `ctrl+c`       |
| 粘贴   | `ctrl+v`       |
| 剪切   | `ctrl+x`       |
| 撤销   | `ctrl+z`       |
| 反撤销 | `ctrl+shift+v` |
| 全选   | `ctrl+a`       |

## 提升速度型①

| 说明                   | 快捷键           |
| ---------------------- | ---------------- |
| 智能提示               | `alt+enter`      |
| 使用块环绕             | `ctrl+alt+t`     |
| 调出生成generate构造器 | `alt+inster`     |
| 复制指定行             | `ctrl+y`         |
| 切换下一行代码空位     | `shift+enter`    |
| 切换上一行             | `ctrl+alt+enter` |
| 自动生成返回值变量     | `ctrl+alt+v`     |
| 提示代码模板           | `ctrl+j`         |
| 方法形参列表提醒       | `ctrl+p`         |
| 向上/下移动代码        | `ctrl+shift+↑/↓` |
| 向上/下移动一行        | `alt+shift+↑/↓`  |

## 提升速度型②

| 说明                           | 快捷键         |
| ------------------------------ | -------------- |
| 批量修改指定变量名/方法名/类名 | `shift+F6`     |
| 抽取代码重构方法               | `ctrl+alt+m`   |
| 重写父类方法                   | `ctrl+o`       |
| 实现接口                       | `ctrl+i`       |
| 选中结构大小写切换             | `ctrl+shift+u` |
| 批量导包                       | `ctrl+alt+o`   |
| 格式化代码                     | `ctrl+alt+l`   |

## 快速生成型

| 说明                  | 快捷键             |
| --------------------- | ------------------ |
| 生成注释              | `ctrl+`/           |
| 块注释                | `ctrl+shift+/`     |
| System.out.print()    | `sout + enter`     |
| 快速生成表达式 **注** | `.var+enter`       |
| 输出某变量            | `变量.sout +enter` |
