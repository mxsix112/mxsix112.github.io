# 英菲克 DR01 开箱


<!--more-->

> 因为之前的鼠标在使用的时候鼠标滚轮出现了断裂现象因此最近从网上筛选了一些鼠标。我的需求是静音，多dpi，无线（2.4g）其他
> 配置则对于我来说没有什么多大的吸引力了。综合选择我最终将目标放在了 英菲克DR01 上。



## 外观

### 📦包装 

简约，快递盒子+英菲克硬纸盒+包装盒，内部一根约长27cm的type-c数据线，一个鼠标及一份说明书。

### 🖲️外观 

整体为黑色，表面摸起来比较顺滑，左高右低，白色电量灯在使用时是常亮白色无法关闭。

## 使用体验

### ✅优点

1. **先进的type-c接口**
2. 多键位，拥有侧键，左右键静音微动，
3. 内置**500**mha电池，三模(有线充电情况下依旧可以使用)连接
4. 三档位dpi(800-1000-1600)自由切换
5. 电量显示，价格相对来说便宜，自动休眠(3秒小休，10分大休)



### ❎缺点

1. 摸起来轻飘飘的不舒服；
2. 底部的防滑贴有点太滑了；
3. 六个键体只有左右键是静音微动；
4. 侧键位用起来很拉跨不太舒服，靠近滚轮点击可能会出现类似空腔音?

### ✍️总结

总体来说中规中矩，鼠标回报率是125hz，对于我来说不打游戏应该是足够了。之前用的是冰狐q11鼠标，也是无线静音游戏鼠标，不过配置啥的忘完了，毕竟是四年前的产品了，虽然是二手电池重新包装但是使用起来非常的耐用，表面也有一些涂层处理，用起来很爽很很舒服可惜滚轮断了。因此才换了这个dr01，价格方面来说二者没多大区别，使用起来对于我个人而言还行。当然有很多的静音鼠标去选择，看个人的喜好吧。
